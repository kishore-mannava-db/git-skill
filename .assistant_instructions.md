# Assistant Instructions

## Custom Skills
When asked about data quality, validation, completeness, duplicates, or audit workflows, use the git-skill skill.
When asked about financial SQL, trial balances, aging analysis, reconciliation, or KPIs, use the git-skill skill.

## General Guidelines
- Always use DECIMAL(18,2) for financial amounts, never FLOAT
- Standardize timestamps to UTC
- Include row counts in query results
- Round numeric values to 2 decimal places
